Integrate NyroModal  jquery plugin into your blogs. Open Images and Native Galleries on the same window without touching the html code.

== Description ==

This plugin allows to use the NyroModal features for images and wordpress galleries without touching the html. 
It includes an option "Open in the same window" on the edit panel before inserting the image on your post. 

For the native wordpress galleries the plugin inserts the class name automatically.

== Installation ==

1. Upload ‘wp-nyro’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. And that's it.
